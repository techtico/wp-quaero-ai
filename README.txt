== Description ==

WP Quaero AI plugin is a powerful and efficient tool that enhances website search functionality using the latest advancements in artificial intelligence. 
With this plugin integrated into your website, users can now enjoy a seamless and highly optimized search experience like never before.

Key Features:

*   Seamless Integration
*   Advanced AI-Powered Search Algorithm
*   Short Description Extraction
*   Related Links Generation
*   Real-Time Indexing and Updates

== Installation Guide ==

1. Upload `wp-quaero-ai` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the API Key & Bot Id from the Quaero AI App
4. Sync the post & posts to Quaero AI App
